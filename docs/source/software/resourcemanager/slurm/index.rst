SLURM
=====
This is a **user guide**, it does not matter if you have or not some experience 
using Slurm, here you will find some useful examples and commands. We try to 
cover our most common use cases.

.. note::
    This guide assumes a basic comprehension about Slurm. If you don't have
    any idea of what Slurm is or what things it does in the HPC world, we
    strongly encourage reading some of the articles describe in `more information`_
    section.
