Grafana
========

Grafana allows you to query, visualize, alert on and understand your metrics no matter where they are stored. [1]_

Grafana is an open source metric analytics & visualization suite. It is most commonly used for
visualizing time series data for infrastructure and application analytics but many use it in
other domains including industrial sensors, home automation, weather, and process control. [2]_
