Tested on (Requirements)Â¶

  • OS base: CentOS (x86_64) \(\boldsymbol{\ge}\) 6.6

  • Compiler: GCC \(\boldsymbol{\ge}\) 5.4.0.

  • Requirements:

    • Fortran NetCDF \(\boldsymbol{\ge}\) 4.4.3 with GCC

    • MPICH \(\boldsymbol{\ge}\) 3.2 with GCC

    • zlib \(\boldsymbol{\ge}\) 1.2.11 with GCC

    • JasPer \(\boldsymbol{\ge}\) 1.900.1 with GCC

Compile WSPÂ¶

The WRF Preprocessing System (WPS) 1 is a set of three programs whose collective role is to prepare input to the real.exe program for real-data simulations.

  1. Download the latest version of WSP

    wget https://github.com/wrf-model/WPS/archive/v3.7.tar.gz
tar vf 3.7.tar.gz
cd WPS-3.7

  2. Load the correspondent modules and execute the configuration script.

    module load jasper
./configure

  3. Edit the configuration file configure.wps

    WRF_DIR = path/to/wps

// Depends on your compiler version you should remove -f90=ifort from the following line
DM_FC = mpif90 -f90=ifort

  4. Compile it.

    ./compile | tee wps-compilation.log
